# Tourism Simulation

## Overview

It is an simple simulation project of tourist traffic in Krakow.

## How to run

This project requires:
- Python 3.6.3
- Pip

Download Python from:
https://www.python.org/downloads/

Run following script:
```bash
sudo ./launch.sh
```
